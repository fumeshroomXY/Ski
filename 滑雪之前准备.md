# 天气篇
- 大雪天气还是不要去滑雪了
  - 滑雪时可能出现各种情况，比如视线不清，身体和头部很冷等
  - 往返路中也可能出现各种情况，比如交通事故，道路封路，堵车等 

# 装备篇
- 滑雪时还是需要防雪防水滑雪服，摔倒或者坐缆车时不易进雪弄湿
- 雪镜起雾如何解决
  - 原因是人体产生的热气进入雪镜然后遇冷起雾
  - 雪镜要么完全戴好，要么完全摘下来，不要扣在头上，让头上的热气进入雪镜
  - 面罩（口罩）不要盖住鼻子，以免呼出的热气上升进入雪镜；戴面罩可以漏出鼻子和嘴巴；或者选择带呼吸阀的口罩
  - 戴帽子或头盔时不要压住雪镜上下通风口，可以买带风道的头盔或留一指缝
  - 在外滑雪时起雾也不要立刻擦雪镜内面，越擦越花；在外面等一会，等雪镜内外的空气流通之后，起雾会缓解很多
  - 假如摔跤后雪镜不可避免的进水了，在餐厅可以用纸巾沾雪镜内面水渍，不要擦，否则会擦坏防雾层
  - 早上出发可以
- 雪镜选择（可开窗，鼻梁贴合，可变色，加热）

# 穿鞋
- 袜子捋平
- 将鞋放在地上穿，**拉开鞋舌**再穿
- 穿进之后，**向下磕一下**，让脚压到后面去
- 绑扣从上面两个扣开始，不要一次性绑太紧；
- 穿好鞋子后，走路时尝试**屈腿压一压鞋子鞋舌**
- 滑雪时绑扣可以根据感觉随时调整松紧

# 穿雪板
- 将雪鞋底部在雪板卡扣上蹭一蹭，把底部的雪蹭掉，更容易穿雪板
- 确认雪板卡扣是未翘起状态，再穿雪板（有时卡扣是翘起状态，这时是塞不进雪鞋的）
